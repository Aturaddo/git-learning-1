# git-learning-1
Aprendendo a utilizar o Git e GitHUB

## I'm learning...
```
git init - Inicializar um novo repositório.

git add  ou . (Todos os arquivos) - Add aquivos alterados.

git status - Verificar status do Repositório.

git commit -m "mensagem: - Commmit com uma mensagem.

git push - Subir para o GitHUB.

git branch - Verificar Branchs.

git branch  - Criar uma Branchs.

git checkout  - Alternar branch de trabalho.

git checkout -b <branch-origin> <new-branch> - Alternar branch de trabalho e criar uma nova branch.

git pull - Trazer atualizações ativas dos Arquivos.

git merge  - Unir Branch na principal, para isso deve ir na Branch que deseja unir e digitar esse comando.
